This software is used to do the rejuvenation for android mobile phone and include below function:
1. Memory status:
           Application will show the total/ free memory and memory usage information on screen
2. Rejuvenation strategy level one had realized - Done
           When memory usage over the threshold, level 1 rejuvenation strategy will be load which killing background/ unused process to free memory.
3. Rejuvenation strategy level two - Doing
           When strategy level one do not take effect to free memory, rejuvenation strategy level two will be load to reboot android device. But some bugs appears with below log.
